xssValidator for Burp

xssValidator 插件可以增强 burp 对 xss 漏洞的检测。

安装

测试

运行 xss-detector 服务

xss.js 文件中包含 XSS-Detector 服务的监听 IP、端口，默认为 127.0.0.1:8093。

如果没有修改 xss.js 文件中的配置，那么xssValidator 设置保持默认即可。

Intruder 中使用 xssValidator Payloads

这里用 dvwa 中的 XSS(Reflected) 来测试。发送请求，拦截数据包并发送到 Intruder，设置 Payload 位置。

Payloads type 选择 extension-generated，generator 选择 xssValidator Payloads。

添加 xss 测试成功的 flag，复制至 grep -match

然后 start attack 开始 xss 盲打攻击。

开始攻击后，xss-detector 服务会接收到请求。

匹配到 flag(fy7sdufsuidfhuisdf) 即为存在 XSS 漏洞。

可以把 payload 复制出来在浏览器进行请求来验证。

验证成功。
